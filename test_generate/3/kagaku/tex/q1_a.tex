\section*{1. 電解質と非電解質}
\begin{enumerate}
    \item 水に溶かしたとき、水溶液に電流が流れる物質を（ \textcolor{red}{\textbf{電解質}} ）、流れない物質を（ \textcolor{red}{\textbf{非電解質}} ）という。
    \item 次のア～オの物質から、(1)の物質をそれぞれ \textbf{すべて} 選び、記号で答えなさい。\\[0.2cm]
    ア. 塩化ナトリウム \hspace{0.5cm} イ. 砂糖（ショ糖） \hspace{0.5cm} ウ. 塩酸 \hspace{0.5cm} エ. エタノール \hspace{0.5cm} オ. 塩化銅\\[0.2cm]
    流れる（ \textcolor{red}{\textbf{ア、ウ、オ}} ） \hspace{1cm} 流れない（ \textcolor{red}{\textbf{イ、エ}} ）
\end{enumerate}
